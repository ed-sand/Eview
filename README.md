# Eview
Visor de imagenes en Java

Uso la linea de comandos (Ubuntu Linux 17.04 x64) para compilar y ejecutar el programa.

INSTRUCCIONES:


    git clone https://github.com/ed-sand/Eview.git

    cd Eview/src/com/edsand/

    javac Eview.java

    java Eview
